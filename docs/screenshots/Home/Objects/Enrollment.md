## Enrollment

### Purpose

Tracks student enrollments.

### Fields

| Field Name          | Data Type | Description |
|------------         |-----------|-------------|
| Enrollment Date     | Date | Enrollment date |
| Status | Picklist | Active, Pending, Completed |
| Payment Status | Picklist | Paid or Unpaid |

### Relationships

- Linked to Student
- Linked to Batch
